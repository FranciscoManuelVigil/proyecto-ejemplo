\# Proyecto Ejemplo – Calculadora Básica

Este proyecto forma parte de la práctica de Git y GitHub.

Aquí construiremos una calculadora en Python de forma incremental, aplicando

técnicas de pair programming y control de versiones.

\## Estructura

\- `src/main.py`: Programa principal que muestra un pequeño menú.

\- `src/operaciones.py`: Funciones matemáticas básicas.

\- `src/\_\_init\_\_.py`: Archivo para indicar que `src` es un paquete de Python.

\## Objetivo inicial

El proyecto comienza con funciones básicas de suma, resta, multiplicación y

división, listadas pero aún no implementadas.
